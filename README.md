# Calculadora Científica en Java

Este proyecto es una *Calculadora Científica desarrollada en Java, utilizando el IDE NetBeans y control de versiones con *Git y GitHub*.  
El proyecto se trabajó con enfoque en la metodología *Scrum*, creando un MVP (Producto Mínimo Viable) funcional.

Este MVP incluye operaciones básicas y varias operaciones científicas como potencias, raíces, Multiplicación, División, etc.

## Requisitos

Para ejecutar este proyecto necesitas:

- *Java JDK* (versión JDK 20).
- *NetBeans* (versión 27).
- *Git* instalado en tu computadora.
- *Acceso a una cuenta de GitHub*.

## Clonar el proyecto

1.      Abrir NetBeans.
2.	Ir al menú: Team → Git → Clone…
3.	Pegar la URL del repositorio del proyecto de GitHub.
4.	Hacer clic en Next y seguir los pasos del asistente.
5.	Esperar a que NetBeans termine de clonar el repositorio.
6.	Cuando finalice, seleccionar Open Project para abrirlo en el IDE.

##  Ejecutar la calculadora

1.      Abre NetBeans y da clic derecho sobre el proyecto → Set as Main Project (si es necesario).
2.	Asegúrate de que no haya errores.
3.	Abre la clase principal (la que contiene el método main).
4.	Algunas formas de ejecutar:
	  • Menú superior Run → Run Project.
	  • Clic derecho en el proyecto → Run File.
	  • Presionar el botón verde Ejecutar.
5.	Probar las funciones:
	  • Operaciones básicas.
	  • Funciones científicas.
	  • Limpieza de pantalla.

## Funcionalidades actuales del MVP

1.      Operaciones básicas:
          • Suma
	  • Resta
	  • Multiplicación
	  • División
	  • Porcentaje
	  • Cambio de signo (+/-)
	  • Borrar dígito (C)
	  • Borrar todo (CE)
	  • Igual (=)
2.      Operaciones  científicas:
          • Paréntesis: ( )
	  • Potencia (^), x²
	  • Raíz cuadrada (√x)
	  • Raíz n-ésima
	  • Exponencial (eˣ)
	  • Logaritmo natural (ln)
	  • Constante π
	  • Funciones trigonométricas:
	    • sin, cos, tan, cot
	    • sec, csc
	  • Funciones trigonométricas inversas:
	    • arcsin, arccos, arctan
3.      Funciones de memoria:
          • MS – Guardar en memoria
	  • MR – Recuperar memoria
	  • MC – Limpiar memoria
	  • M+ – Sumar a memoria
	  • M- – Restar a memoria
4.      Interfaz:
          • Pantalla amplia para visualizar operaciones.
	  • Botones organizados por colores según la función (básicas, científicas y memoria).
	  • Diseño tipo “tablet” fácil de usar.
